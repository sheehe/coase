FAQ

Answers before you ask

Do I need an API key?
Yes. Coase AI talks to external LLMs — DeepSeek, OpenAI, or Anthropic. Any one works. We recommend running with two providers from different families so the Reviewer agent can give you a genuinely independent read.
Where does my data live?
Entirely on your machine, in the local data/ folder. Nothing is uploaded anywhere except the natural-language prompts and tool outputs that go to the LLM APIs you chose. Raw datasets never leave your disk.
Can it run offline?
The R execution layer is fully local and offline. But the four agents depend on LLM API calls, so you need an internet connection while the pipeline is running.
How is this different from ChatGPT or Claude chat?
A chat UI is a conversation — you drive every step. Coase AI is an automation loop: the agents design the study, run the code, and critique their own output. You sit at the checkpoints, not at the keyboard for every prompt.
